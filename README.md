# 🚲 US Bikeshare Data Analysis

## 📄 Project Description
In this project, Python is used to explore data related to bike share systems for three major cities in the United States: Chicago, New York City, and Washington. The script provides an interactive experience in the terminal, allowing users to filter the dataset and compute descriptive statistics to uncover bike share usage patterns.

## 🛠️ How to use
To run this project, you will need Python installed on your machine, along with the `pandas` and `numpy` libraries.

1. Clone the repository.
2. Ensure you have the dataset files (`chicago.csv`, `new_york_city.csv`, `washington.csv`) in the same directory as the script.
3. Run the script in your terminal:
   ```bash
   python bikeshare.py
